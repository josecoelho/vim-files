

## Quickstart

```
$ git clone git@github.com:josecoelho/vim-files.git ~/.vim
$ mkdir ~/.vim/bundle
$ curl https://raw.githubusercontent.com/Shougo/dein.vim/master/bin/installer.sh > installer.sh
$ sh ./installer.sh ~/.vim/bundle/dein
$ rm ./installer.sh
```

### Using NeoVim

```
  ln -s ~/.vim ~/.config/nvim
```
